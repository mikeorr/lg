Graphical Programs(part II)
MindsEye:

MindsEyeis a project to develop a GPL 3D program. It´s main features are modular design, multi scene/multi user concept( where different people interact or watch a scene), vision of "kernel-system" instead of "modeler-system", object oriented modeling/designing and network support as MindsEye-kernel.

Current version of the program is 0.5.36, but it´s already functional. You can find all the documentation here and also its newsgroup
