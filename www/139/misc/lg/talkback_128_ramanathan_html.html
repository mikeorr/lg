Installing SVN is pretty easy. There are pre-built binary packages available for most of the popular distributions. Check out: http://subversion.tigris.org/project_packages.html for package links for your distribution.

If you want to install from source: http://svn.collab.net/repos/svn/trunk/INSTALL has instructions.

For more details on installing/maintaining a SVN server check out the free SVN book at: http://svnbook.red-bean.com/

Hope this helps.
