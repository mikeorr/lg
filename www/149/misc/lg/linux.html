The best setup for learning to use linux:
	1. Have one screen open on man pages/info pages
	2. Have one screen open on /usr/share/doc/
	3. Have one screen open on tldp.org
	4. Have one screen open on LG (!)
	5. Start reading/playing/hacking
DONT: Login as root except to install packages.
DO: Ignore the above injunction as soon as you feel that breaking your
system and fixing it are part of step 5.
 
If you use Debian, then 1/2/3 can be combined/enhanced by installing
 
	dwww, dpkg-www, doc-linux-((text)|(html))
 
and a web browser.

There must be something similar for other distros. Perhaps there should be a move to create lsb-doc[1]!

Regards,

Kapil.

[1] LSB stands for Linux Standards Base which describes how different distributions should keep things compatible/equivalent to other distributions. --
