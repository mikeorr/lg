Sure, just run two copies of whatever it is you want on each screen. Note that without things like Xinerama support, you won't be able to move windows between screens, but then this can't be a requirement if you don't want the mouse to move between them either. For instance: 

exec fvwm -d :0.0
exec fvwm -d :0.1
Now, assuming you setup X just to use two monitors without enabling things like Xinerama, such operations tend to happen automatically in my experience, as the above example with FVWM. I couldn't say how GNOME or KDE works in that regard.
